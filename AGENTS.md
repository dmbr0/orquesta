## Elixir & Erlang Code Standards

All Elixir and Erlang code in this project must conform to the following rules:

- **Specification:** `.kilocode/rules/spec.md`
- **Elixir style:** `.kilocode/rules/elixir-style.md`
- **Erlang style:** `.kilocode/rules/inaka-erlang.md`
- **Safety-critical coding rules:** `.kilocode/rules/nasa-elixir.md`

Read all four files before writing or modifying any Elixir or Erlang code. Do not submit code that violates any rule in these documents.
